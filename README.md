# ExpressAPI

**Description:**
Created an express server with the following endpoints; "/ping", "/chuck", "/count", "/reset", "/sum", and "/subtract".

**How to Install and Run:**
_(Using the Terminal)_

1.  git clone {repoURL}
2.  cd into folder
3.  yarn install
4.  yarn dev
